# PureGlowPHP
Tienda en línea de cosmeticos como Mercado Libre. Como vendedor, sube tus productos, modificalos, envia mensajes a compradores y otros vendedores, como usuario, compra, guarda en favoritos, habla con vendedores y modifica carrito. 

Proyecto en colaboración, creé el diseño de la base de datos en mysql y los comando request dentro de la página, además de trabajar en las funcionalidades en el menú de comprador y la vista y las funcionalidades de la página de compra de producto. La integración del Api de Paypal y la página de puntuación con su funcionalidad.

Lenguajes usados: html, javascript, sql, css, java, php
